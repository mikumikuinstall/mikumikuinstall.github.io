Welcome to MikuMikuInstall!

MikuMikuInstall is a simple installer for MikuMikuDance (MMD). It will install the latest version and all of its dependencies. It also includes the latest version of MikuMikuEffect.

What is MikuMikuDance?

MikuMikuDance (MMD) is a free 3D animation software developed by Yu Higuchi, that lets you import and animate 3D models, even if you have no prior 3D animation experience. MMD is used by many people to create animations of the Vocaloid character Hatsune Miku, but it can be used to animate any 3D model.

For more information, visit the MikuMikuDance Wiki.

What is MikuMikuEffect?

MikuMikuEffect (MME) is a free plugin for MikuMikuDance (MMD) that lets you add advanced visual effects to your renders. It is commonly used to add specialised shaders to MMD videos.

For more information, visit the MikuMikuDance Wiki.
